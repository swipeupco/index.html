Click "Copy Signature" then paste directly into Gmail, Outlook, or Apple Mail signature settings.

How to Install

1
Fill in your details on the left — the preview updates live.
2
Gmail: Settings → See all settings → Signature → Create new → Paste.
3
Outlook: File → Options → Mail → Signatures → New → Paste.
4
Apple Mail: Mail → Settings → Signatures → + → uncheck "Always match font" → Paste.
